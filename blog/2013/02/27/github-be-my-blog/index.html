利用Github搭建个人博客

别了，各种虚拟主机

本文将详细介绍如何在Windows系统下利用一个很酷的博客框架octopress搭建博客，并发布到GitHub Pages上，以及后期博客发布后的更新工作….

怎么搞？

详细请参考下面d教程

  1. Github上搭建博客
  2. Blog = GitHub + Octopress
  3. Hello Octopress

常用重要的命令

  1. 生成静态文档

    sh $ rake generate

  2. 本地预览http://localhost:4000

    sh $ rake preview

  3. 部署到github

    sh $ rake deploy

  4. 部署完毕后提交source分支

    sh $ git add . $ git commit -m "commit msg" $ git push origin source

  5. 创建新文章YYYY-MM-DD-hello-world.markdown

    sh $ rake new_post['hello world']

    相关链接:

    • markdown语法
    • GFM
    • Sublime text markdown文档预览插件

  6. 更新主题.themes/xxx

    sh $ rake update_source['xxx'] $ rake update_style['xxx']

  7. 查看更多rake 命令

    sh $ rake -T
