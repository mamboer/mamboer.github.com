如何在GFM中高亮代码呢？

举个例子：

``` js
var x = 1;
var fun = function(){};
```

请参考这里，GFM高亮的语言别名

例如AppleScript的别名(aliases)为osascript，那么高亮AppleScript应该写成：

``` osascript
//代码内容
```

Octopress里面如何使用类似GFM的代码高亮？

Octopress是支持类似GFM的代码高亮的，通过内置的Backtick Code Blocks

如果在rake generate时发生pygments相关的错误，请检查：

  1. 有没有装python
  2. 版本是不是2.7.x
  3. 如果不是，需要安装2.7.x版本的python
  4. 安装完毕后，把python.exe所在的目录添加到环境变量中去
  5. 关掉所有的git bash然后重新打开bash，cd到octopress目录再次rake generate命令
