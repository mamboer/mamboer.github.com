Clipboard API and events 里面描述了剪切板事件onpaste，通过onpaste可以访问剪切板数据，然后使用blob、和FileReader方法将剪切板数据转为dataurl即可。

类似QQ截图，然后直接粘贴到网页上传。非常方便。
