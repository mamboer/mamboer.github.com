本文摘译自Why Web Developers Should Learn Web Design。

码农除了会码还该懂点设计

做码农其实蛮酷的，做会设计的码农那才是真正的牛牪犇逼。

  1. 提高视觉设计技能

    学习设计会让你沉浸于不同的视觉设计工具和设计理念中去。

  2. 提升网页设计品味

    如果码农学点设计，可以灰常轻松地辨别网页视觉表现上的好与坏，提升审美能力。

  3. 更具竞争力

    学习设计会帮助码农更多地了解产品的商业诉求。

  4. 让设计师泪奔

    又会码又会设计，牛逼的话你可以和你老板说你的项目可以不用设计了。你一个人搞定一切！！

学习设计的一些贴士

  1. 寻找设计灵感

    没事去围观下别人的优秀的设计作品，最终提升自我的设计品味。推荐的一些站点：

    • Dribble
    • Best Web Gallery

  2. 找些设计教程做做

    练习是技能提升的基础！推荐两个有此类教程的站点：

    • 1stwebdesigner
    • Webdesign Tutplus
